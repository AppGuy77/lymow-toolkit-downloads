# Lymow Toolkit

Local companion dashboard for the Lymow One robotic mower — live map with accurate coverage paint,
per-zone cutting settings, scheduler, calendar, freshness overlay, RTK heat maps, fault
auto-recovery, mow history, and remote control.

Sign in with your Lymow account after starting the add-on: click **Open Web UI**.

Full instructions are on the **Documentation** tab.
